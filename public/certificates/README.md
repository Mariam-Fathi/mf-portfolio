# Certificate images

Add your certificate images here. Expected filenames:

- `time-series.png` — Time Series (Kaggle)
- `data-engineering.png` — Data Engineering (DeepLearning.AI)
- `computer-vision.png` — Computer Vision (Kaggle)
- `ai-agents.png` — AI Agents (Kaggle × Google)

You can use `.png` or `.jpg`. If you use different names, update the `image` path in `src/components/Certificates.tsx` for each certificate.
